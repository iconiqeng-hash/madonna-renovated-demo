# Madonna Salon — Premium Website Renovation

A luxury renovation of the Madonna Salon website built with React.js and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Replacing Images

Replace placeholder images in `src/data/content.js` with your actual salon images from madonnasalon.in.
